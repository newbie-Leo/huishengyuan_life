#### 惠生园出行指南

- 惠生园（以及通惠家园其他小区）是个神奇的地方
- 几乎每个次打车都能走丢
- 地图上相隔一米的地方也要步行很久找到楼梯
- 开车出小区一定要小心翼翼，错过一个路口就去了不同的世界
- 小区里都在问怎么出去...
- 小区外都在问怎么进来...
